Compare-at Price Mismatch rule: only evaluated when Duplicate = Yes; a listing having a compare-at price and another having none counts as a mismatch, same as two different non-null compare-at values.

Data source: Shopify Admin GraphQL API (read-only) · store ledsone.de (ledsone-de.myshopify.com) · fetched via a server-side serverless function (/api/sukirtha-req3-slow-moving-stock) using a Shopify Admin access token stored only as a Vercel server-side environment variable — never present in this HTML or any client-side script.
Update frequency: not a scheduled/cached snapshot — this page calls Shopify live every time it loads. "Last checked" reflects the actual retrieval timestamp of that call; the 90-day window is calculated from the retrieval date, not a fixed historical date.
Data grain: one row per Shopify Variant ID. Units Sold is joined to orders via the line item's Variant ID, never SKU alone.
Units Sold (90d) rule: sum of each eligible line item's refundableQuantity (the quantity net of any already-refunded units for that line item) across all orders in the 90-day window, excluding cancelled orders (cancelledAt set), test orders (test:true), and voided orders (displayFinancialStatus:VOIDED).
Current Stock rule: Shopify inventory available quantity, summed across the operational location(s) returned for that variant (ledsone.de currently has a single active location, LEDSone DE LTD). If a variant is not inventory-tracked, Current Stock and Days of Stock show "Not Tracked"/"Not Assessable" — never treated as zero.
Status rule (strict): Slow-Moving = Units Sold (90d) < 10 AND Current Stock > 100. Not Assessable = inventory not tracked. Otherwise OK.
Governance: Shopify remains the sole authoritative source for products, variants, inventory, and orders. This dashboard is a read-only analytical view; no product, variant, price, inventory, or order record was created, changed, or merged by this page or its build process.
