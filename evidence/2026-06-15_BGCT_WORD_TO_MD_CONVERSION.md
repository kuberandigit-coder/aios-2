# BGCT Word Documents → Markdown Conversion

**Date:** 2026-06-15
**Owner:** Kuberan

## Objective
Convert BGCT Word documents (.docx) into Markdown (.md) files so they can be added as **source material for ChatGPT projects**.

## Background
ChatGPT projects ingest reference/source files best as clean Markdown. The BGCT material existed only as Word documents, which are not ideal as project sources. Converting them to `.md` makes the content easy to attach, search, and reference inside ChatGPT project context.

## Work Done
- Converted the BGCT Word documents to Markdown format.
- Prepared the resulting `.md` files for use as ChatGPT project source/reference material.

## Outcome
- BGCT content now available as Markdown source files for ChatGPT projects.
- Source material is easier to attach and reference within the relevant ChatGPT project context.

## Future Considerations
- Keep the Markdown versions in sync if the source Word documents are updated.
- Store the converted `.md` files alongside the relevant project workstream for easy reuse.
